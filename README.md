# 🧠 치매 환자 수 vs 요양시설 정원 분석 프로젝트

> 공공데이터를 활용하여 지역별 노인 인구, 치매 환자 수, 요양시설 정원을 비교하고 지역별 수요-공급 불균형을 분석한 프로젝트입니다.

---

## 📖 프로젝트 소개

이 프로젝트는 **전국 노인 인구 대비 추정 치매 환자 비율**을 분석하고,  
**추정 치매 환자 수 대비 요양시설 정원**을 비교하여  
지역별로 **요양시설이 부족한지, 적정한지 판단**하는 데이터 분석 프로젝트입니다.

---

| 기술 | 용도 |
|------|------|
| Python | 전체 분석 및 실행 |

---

## 🎯 분석 목표

- 노인 인구 대비 추정 치매 환자 비율은 어느 정도인가?
- 추정 치매 환자 수 대비 요양시설 정원이 충분한가?
- 어떤 지역이 요양시설이 부족한가?
- 정책적으로 어디에 시설 확충이 필요한가?

## 📊 사용 데이터

| 데이터 | 설명 |
|------|------|
| facility.csv | 요양기관 위치 및 정보 |
| resident.csv | 요양기관 정원 및 현원 |
| dementia.csv | 지역별 노인 인구 수 및 추정 치매 환자 수 |

---

## ⚙️ 주요 처리 과정

### 1. 데이터 전처리

- 주소 → `시도 + 시군구` 단위로 변환
- 행정구역 통일
  - 전라북도 → 전북특별자치도
  - 강원도 → 강원특별자치도
- 성별: 전체만 사용
- 연령: 지정된 6개 구간만 사용
  - 60~64세
  - 65~69세
  - 70~74세
  - 75~79세
  - 80~84세
  - 85세이상
- 정원이 0이고 현원이 존재하는 경우, 현원 값을 정원으로 대체

---

### 2. 데이터 병합

- `장기요양기관코드` 기준으로 facility + resident 연결
- 지역별 요양기관 정원 및 현원 합산
- dementia 데이터에서 지역별 노인 인구 수와 추정 치매 환자 수 집계

---

### 3. 핵심 지표 계산

- 노인인구수
- 치매환자수
- 노인대비 치매비율
- 총정원
- 총현원
- 수용률 = 총정원 / 치매환자수
- 정원차이 = 총정원 - 치매환자수
- 부족인원 = 치매환자수 - 총정원

```text
총정원 ≥ 치매환자수 → 적당
총정원 < 치매환자수 → 부족
```
